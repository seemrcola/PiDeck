# 更新日志

[English](CHANGELOG.md)

这里记录 pi-desktop 各版本的重要变化。

## v0.4.5 - 2026-06-05

### 新增
- 配置管理支持导出/导入功能：可将 models.json、auth.json、settings.json
  三个配置文件打包为 JSON 文件备份，方便恢复和迁移。
- Provider 配置新增兼容性设置：可视化编辑 supportsDeveloperRole 和
  supportsReasoningEffort 选项，无需手动修改 JSON。
- 输入框图片支持点击放大预览：附加图片后点击缩略图可查看原图。
- 文件抽屉顶部显示当前会话 agent 修改过的文件列表。
- 修改文件列表支持右键菜单：打开文件、在文件夹中显示、引用到输入框。
- 会话结束后在状态栏显示总耗时（如 3.2s / 1m23s）。
- Reload/Restart 按钮增加 loading 文字和禁用状态，避免重复点击。

### 修复
- 错误检测逻辑优化：避免将正常工具输出误判为错误消息（如 "Successfully
  replaced" 等成功提示不再显示为红色错误）。
- 输入框图片预览区域与 textarea 重叠问题：调整 grid 布局，图片预览
  独占一行，不再遮挡输入框。
- Agent 异常结束时（API 错误等）将错误提示写入会话，避免空白响应。
- 修正 agent_end 错误提取逻辑：遍历 messages 数组查找异常消息，不再依赖固定位置。
- 增大修改文件列表字体和颜色对比度，提升可读性。
- Git 分支选择器仅显示本地分支，去掉远程分支。

### 优化
- 配置弹窗 UI 优化：宽度增至 900px，导出/导入按钮与保存按钮风格统一，
  Provider 展开区域增加间距，删除按钮图标统一。
- 关闭按钮颜色加深，提高可见性。
- 移除 Reload 按钮（/reload 无法通过 RPC prompt 正确执行），统一使用 Restart 按钮。

## v0.4.4 - 2026-06-05

### 新增
- 输入框上下键切换历史消息：在 composer 中按上/下方向键，可回溯最近发送过的
  消息（类似 CLI 体验）。
- 用户消息编辑按钮：点击可将消息文本回填到输入框，编辑后重新发送。
- Models 页签 API 类型改为自定义下拉选择：预设 openai-completions、
  openai-chat-completions、openai-responses、anthropic、google-generative-ai，
  同时保留自定义输入兼容未知类型。

### 优化
- 配置管理弹窗 UI 整体美化：Models/Auth 展开卡片、表单控件、模型列表面板等
  与主界面风格统一，圆角、浅边框、聚焦绿色描边。
- Agent 启动/重启不再阻塞其他 agent：去掉全局 agentLoading 遮罩，改为按
  当前活跃 agent 的 status 判断，切换项目不再等待。
- 配置保存后不再自动 reload，用户可通过 Restart 按钮手动重载。
- Agent 响应中禁止切换 model 和思考级别，按钮自动禁用避免中途误操作。
- 工具调用状态修复：改为检查最后一条工具消息的状态，工具组完成后不再显示
  "工具调用中"。
- 思考气泡位置恢复至消息列表底部，流式输出时按时间顺序自然堆叠。

## v0.4.3 - 2026-06-04

### 新增
- 思考过程实时显示：模型推理时展示可折叠的思考内容，避免用户感觉“卡住”。
  思考内容持久化到消息中，当前会话和历史会话均可查看。
- RPC 日志面板：右键 agent 标签页打开，展示详细的请求/响应/事件流，
  点击可展开查看完整 JSON 数据。
- 设置页新增 DevTools 开关按钮，方便调试。

### 优化
- 设置弹窗宽度从 420px 增加到 640px，提升可读性。
- 清理思考内容中的 ANSI 转义码（终端颜色序列如 `\x1b[38;2;...m`）。

## v0.4.2 - 2026-06-04

### 新增
- Agent 忙碌时消息自动排队：正在输出时发送消息会进入本地队列，
  agent 空闲后自动逐条发送（带 steer 语义，对齐 pi CLI 行为）。
- 排队消息支持单条取消，点击气泡上的"取消排队"即可移除。
- 排队消息 UI：半透明虚线气泡 + 旋转排队图标 + 发送按钮"排队发送"脉冲动效。

### 优化
- 切换 agent 时排队消息按 agentId 隔离，不会误发到其他 agent。
- 发送失败的消息回退到排队队列并 toast 提示，避免丢失。
- 改善 restart 后历史会话加载：自动补全 sessionPath，loadMessages 失败重试。

### 修复
- 修复 agent 结束后排队消息无法触发 flush 的问题（agent_end 后同步推送 runtimeState 重置 isStreaming）。
- 修复 restart 后历史会话空白的问题。
- 修复启动时 get_commands 超时导致控制台报错的问题。

## v0.4.1 - 2026-06-03

### 优化
- 用户消息改为纯文本显示，避免特殊字符被 Markdown 解释导致渲染异常。
- 通知逻辑优化：只在会话结束时发送通知，工具调用时不触发。
- 工具调用期间保持 thinking bubble 动画显示。
- 隐藏项目列表的折叠箭头图标，界面更简洁。
- 减小项目列表左侧空白，布局更紧凑。
- 调整 agent 行关闭按钮位置，避免与边框重叠。

## v0.4.0 - 2026-06-02

### 新增
- 图片支持：从剪贴板粘贴图片（Ctrl+V）或拖拽图片到聊天输入框。
- 用户消息中显示图片，支持点击放大预览。
- 历史会话图片恢复：重新打开历史会话时正确显示之前发送的图片。
- 会话结束通知：agent 完成响应后发送系统通知（可在设置中开关）。
- 大图自动压缩：图片最长边压缩到 2000px，减少模型上下文占用。
- 发送图片到不支持图片的模型时显示错误提示。

### 优化
- 自动将 PNG/WebP 转换为 JPEG 传输，减少图片体积。
- 发送按钮支持仅图片无文字的消息。
- 历史会话加载时从 pi 会话文件中提取并显示图片。

### 修复
- 修复历史会话显示 thinking/reasoning 内容而不是实际回复的问题。
- 修复图片发送失败时无错误反馈的问题（现在显示错误信息）。
- 修复消息摘要中出现 ANSI 转义码的问题。

## v0.3.0 - 2026-06-02

### 新增
- 配置管理弹窗：点击侧栏滑块图标，查看和编辑 pi 全局配置文件（`models.json`、`auth.json`、`settings.json`）。
- Models 页签：可视化编辑器，provider 卡片 + 模型网格列表，支持新增/删除 provider 和模型，内联编辑 id、名称、contextWindow、maxTokens、reasoning。
- Auth 页签：按 provider 查看和编辑 API Key，支持新增/删除 auth 条目，Key 支持显示/隐藏切换和一键复制。
- Settings 页签：key-value 编辑器，自动识别类型（布尔复选框、数字输入框、复杂值 JSON 编辑）。
- 源文件页签：直接 JSON 编辑器，支持切换文件。
- 保存配置后自动 reload（对当前活跃 agent 调用 `agents.reload`）。
- 聊天输入框支持 `!command` 和 `!!command` 执行 bash 命令，与 pi 终端行为一致：`!` 执行并将输出发送给 LLM，`!!` 静默执行。
- Git 分支选择器现在同时获取本地和远程分支，新增分支数量徽章和空状态提示。

### 优化
- 所有 emoji 图标替换为 lucide-react 专业图标（Search、ChevronLeft/Right/Down、Play、Check、GitBranch、Eye/EyeOff、Trash2、Settings、Sliders）。
- 侧栏图标（配置管理 + 设置）使用 distinct lucide-react 图标，hover 高亮。
- Auth 和 provider 表单布局改为横向 label+input 网格，对齐更整齐。
- API Key 输入框在 Models 和 Auth 页签均支持显示/隐藏切换和一键复制。
- 分支下拉 z-index 和 overflow 修复，确保在 chat header 内正常显示。

### 修复
- 修复 chat header 的 Reload 按钮：之前把 `/reload` 当普通消息发给 LLM，现在调用专用的 `agents.reload` IPC 接口。
- 修复配置管理弹窗源文件页签：切换文件时重新加载对应文件内容，不再固定显示 `settings.json`。
- 修复分支下拉因父容器 `overflow: hidden` 被裁剪导致为空的问题。
- 修复 BranchSelector JSX 中的 stray tab 字符可能导致渲染问题。

## v0.1.9 - 2026-06-01

### 新增
- 系统托盘支持：关闭窗口时默认隐藏到系统托盘，设置中可开关。

## v0.1.3 - 2026-05-31

### 新增
- 新增启动时 pi CLI 环境检测，并通过可见弹框展示检测结果。
- 新增通用 pi 命令定位器，适配打包后的 Electron 环境。
- 设置页新增手动检测 pi 环境入口。
- 设置页新增当前版本显示和“检测更新”入口，点击跳转 GitHub Releases。

### 优化
- 打包应用在窗口准备好后再显示，减少启动白屏。

- 未检测到 pi CLI 时，引导用户打开官方安装文档。
